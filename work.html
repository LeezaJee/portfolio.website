This is the client-side interface for the Bollyflix Movie API. The interface is built with React and Bootstrap. Meet App

This is a serverless, progressive web application (PWA) built with React using a test-driven development (TDD) approach with Jest, cucumber and puppeteer. This means every unit and relation between pages are tested before developed. The application uses AWS Lambda to generate a uniquie token to access the Google Calendar API to fetch upcoming events. For data visualization, the library Recharts displays data dynamically in a pie- and a scatter-chart.
